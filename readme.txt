Trashcan Reminder 2.0
Homey will remind you to put the trash out! You can do this by providing your address details. You can also configure a custom schedule if your address is not (yet) supported.
NEW: You can add your bins as device(s). Either one device that shows all dates, or single device per trash type. This also gives you the ability to add multiple addresses!

Supported Cities
A lot of cities are supported with this trashcan reminder. If your city is not supported you can always add your schedule manually or reach out to the community and see if it can be added to the app.

Tags
There are some global tags available with information you can use in your (advanced) flow. If you need some examples on how to configure the app and create the best flows. Please follow the link below and check out the docs!

Widgets
There is one widget available that displays the trash that is collected next. You can configure the height and amount of items to display through the widget configuration.
